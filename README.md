#### Anko 是什么？
Anko 是JetBrains开发的一个强大的库。其目的**在于替代xml，使用代码生成UI布局**， 另外提供了丰富的**属性、方法，避免让开发者写很多重复的代码**，注意一定要去了解其背后的原理。
